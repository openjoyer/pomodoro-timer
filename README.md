# Таймер Pomodoro
Плагин для Visual Studio Code, который позволяет удобно отслеживать время для работы и продуктивности в меню-баре по технике __Pomodoro__

## Функционал
- Команда: **Pomodoro Toggle** (pomodoro.toggle)
- Сочетание клавиш: ``ctrl+D`` (``cmd+D`` на macOS )
- Добавляет Pomodoro таймер на *25 минут* в меню-баре
- Переключение ставит на паузу либо продолжает таймер
- По истечении времени появляется уведомление о завершении таймера
