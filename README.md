# Таймер Pomodoro
Плагин для Visual Studio Code, который позволяет удобно отслеживать время для работы и продуктивности в меню-баре по технике __Pomodoro__

## Описание
- Добавляет удобный Pomodoro таймер прямо в статус-баре Visual Studio Code
- Формат: 25 минут работы - 5 минут отдыха
- Удобное использование по сочетанию клавиш или по клику на таймер

## Использование
- Команда: **Pomodoro Toggle** (pomodoro.toggle)
- Сочетание клавиш: ``ctrl+D`` (``cmd+D`` на macOS )
- Добавляет Pomodoro таймер на *25 минут* в меню-баре
- Переключение ставит на паузу либо продолжает таймер
- По истечении времени появляется уведомление о завершении таймера

## Установка
- ``git clone`` репозитория с GitHub
- Сборка: ``vsce package``
- В настройках расширений: ``Install from VSIX...``

## Используемые технологии
- Язык: Typescript
- Платформа: VS Code
- Дополнительные технологии: npm, node js

## Ограничения
- Фиксированное время сессии (25 минут)
- Работа только в рамках VS Code
